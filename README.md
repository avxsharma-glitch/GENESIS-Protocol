# Genesis Protocol

A single-page cybernetic landing page built with React, Vite, Tailwind CSS, Framer Motion, Lucide, and Three.js.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production output is created in `dist/`.

## Deploy

This project is a static site, so it can be deployed directly to Vercel, Netlify, GitHub Pages, or any static hosting provider.

### Vercel

1. Import the project folder into Vercel.
2. Set the build command to `npm run build`.
3. Set the output directory to `dist`.

### Netlify

1. Connect the project folder to Netlify.
2. Set the build command to `npm run build`.
3. Set the publish directory to `dist`.

## Files

- `src/App.tsx` contains the full landing page.
- `src/main.tsx` mounts the app.
- `src/index.css` defines the global styling and fonts.
